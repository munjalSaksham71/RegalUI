# RegalUI

A powerful tool to create your websites faster


# How to install: 
Just put the link below in your html file and you are ready to rock.

```<link rel="stylesheet" href="https://regalUI.netlify.app/styles.css">```


The documentation Link
[ClickHere](https://regalui.netlify.app/)

 ## Features:

1. ### Typography: 

The Typography component in this library provides a helpful variety of styled text components.
 - heading 1
 - heading 2
 - heading 3
 - Grey Text


2. ### Avatar

The Avatar comes with predefined option that allow you to set its shape, type, color, size, border, and fill.
- Image Avatar
- Image Avatar with border
- Text Avatar

3. ### Alert

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

- Alerts with no icon
- Alerts with icon

4. ### Badges

Badge generates a small badge to the top-right of its child.

- Badges on Avatar
- Text with a badge
- Bigger Text with a badge 
- Badge on button 
- Badge on icon

5. ### Button

Custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

- Primary Buttons
- Secondary Buttons
- Outline Buttons
- Floating Buttons

6. ### Cards

Cards provide a flexible and extensible content container with multiple variants and options.

- Vertical Card
- Horizontal Card
- Text only Card

7. ### Responsive Image

To make image responsive, just use class img-res

- Horizontal Image Responsive

8. ### Input

A basic widget for getting the user input is a text field

- Basic Input Field
- Error Input Field
- Textarea Input Field 

9. ### Lists

Lists are a continuous group of text or image

- Basic-list
- Stacked-list 

10. ### User-rating

Ratings provide insight regarding others' opinions and experiences, and can allow the user to submit a rating of their own.
